Bulma

CSS-ramverket Bulma för att enkelt styla direkt i HTML-koden

Om

Bulma är ett CSS-ramverk för att styla webbplatser. Dess kod finns i CSS, CSS minifierad eller SCSS. SCSS är bra att använda ifall behov finns att ändra i dess ursprungliga styling. Bulma innehåller enbart kod för styling och ingen tillhörande JavaScript följer med. Det finns däremot några JavaScript exempel i documentationen.

Fördelar och nackdelar

Styling av webbsidan gick väldigt smidigt med Bulma. Dokumentationen var strukturerad annorlunda än den hos Materialize så det tog en stund att komma igång. Jag tycker att resultatet blev väldigt snyggt och det finns även stor möjlighet att anpassa utseende. Jag ser inte några större nackdelar, men det hade underlättat med färdiga sidtemplates och att de hade mer egen JavaScript.

Tabeller och formulär

Det var enkelt att implementera formulär och tabeller med Bulma. Man plockar de delar som man vill ha ifrån färdiga komponenter. Jag gjorde dock om tabellen eftersom att den hade otydlig stuktur men den använder Bulmas styling.
